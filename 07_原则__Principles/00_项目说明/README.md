## 重建顺序
1. 运行 `extract_pdf_assets.py` 抽取原书文本和页图。
2. 运行 `build_learning_pack.py` 生成 JSON、配图、Markdown 和说明。
3. 运行 `build_ppts.js` 生成全部 `.pptx`。
4. 运行 `export_ppts_to_pdf.ps1` 与 `render_pdf_to_png.py` 输出校验预览。
