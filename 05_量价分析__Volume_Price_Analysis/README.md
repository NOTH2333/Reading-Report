# 量价分析 / Volume-Price Analysis

这是《量价分析》教学化交付包的项目根目录。

## 主要入口

- `《量价分析》教学化交付包/00_总索引/README.md`：最适合开始阅读的入口。
- `《量价分析》教学化交付包/01_PPT/`：各章 PPT 与章节级生成文件。
- `《量价分析》教学化交付包/02_阅读报告/`：全书阅读报告。
- `《量价分析》教学化交付包/03_关键知识总结/`：按章节知识总结。
- `《量价分析》教学化交付包/04_原书配图素材/`：原书配图与章节素材。
- `《量价分析》教学化交付包/05_中间数据/`：生成过程中的中间数据。

## 工程目录

- `scripts/`：构建脚本。
- `node_modules/`：Node 依赖。
- `package.json` 与 `package-lock.json`：项目脚本与依赖定义。

## 常用命令

```powershell
npm run build:docs
npm run build:ppt
npm run build:all
```
