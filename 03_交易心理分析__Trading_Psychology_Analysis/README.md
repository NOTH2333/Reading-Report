# 交易心理分析 / Trading Psychology Analysis

这是一个围绕《Trading in the Zone》生成学习资料包的项目目录。

## 主要入口

- `deliverables/reports/`：阅读报告。
- `deliverables/summaries/`：按章节整理的知识总结。
- `deliverables/ppt/`：PPT 交付物及对应的 `.js` 生成文件。
- `deliverables/assets/`：交付所需的图示素材。

## 工程目录

- `scripts/`：提取、生成和校验脚本。
- `src/`：源文本或上游内容。
- `pptxgenjs_helpers/`：PPT 生成辅助函数。
- `_work/`：中间产物与检查结果。
- `node_modules/`：Node 依赖。

## 常用命令

```powershell
npm run extract
npm run build
npm run validate
```
