## 顶层结构说明

- `01` 到 `10`：书籍项目主目录，按整理顺序连续编号。
- `scripts/`：仓库级共享脚本，包括 PDF 导出和三本新增书籍的章节化生成脚本。
- `.obsidian/`：Obsidian 工作区配置。
- `.claude/`：工具与代理相关配置目录。

## 重新生成与维护

共享 PDF 导出器仍然可用：

```powershell
python .\scripts\export_book_pdfs.py --validate-only
python .\scripts\export_book_pdfs.py
python .\scripts\export_book_pdfs.py --book-id technical_analysis --force
```

三本新增章节化书籍可单独重建：

```powershell
python .\scripts\teaching_pack_builder.py --book-id antifragile
python .\scripts\teaching_pack_builder.py --book-id black_swan
python .\scripts\teaching_pack_builder.py --book-id echo_of_genius
```

说明：

- `README.md` 是根目录唯一索引页，不再额外维护 `00_库索引.md` 或 `00_书籍精读交付索引.md`。
- 新增三本书均为“一章一个 PPT”的独立交付结构，并保留 `_build/` 以便重建。
- 部分项目目录保留中间素材、脚本和依赖，目的是保证交付物可追溯、可重建。

## 浏览建议

1. 先看本页的“顶层顺序”，确认书籍编号和目录位置。
2. 再进入目标书的项目级 `README.md`。
3. 想直接阅读时，优先打开本页表格中的 PDF。
4. 想看章节演示时，直接进入对应书的 `10_PPT/`。
