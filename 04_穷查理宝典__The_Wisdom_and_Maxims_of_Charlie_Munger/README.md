# 穷查理宝典 / The Wisdom and Maxims of Charlie Munger

这是一个面向阅读、讲解和儿童可读化表达的学习资料包目录。

## 主要入口

- `00_项目说明/README.md`：项目说明与背景。
- `01_阅读报告/`：整本书的阅读报告。
- `02_关键知识总结/`：按章节整理的关键知识总结。
- `03_PPT/`：PPT 交付物及对应 `.js` 源文件。
- `04_配图/`：阅读报告、总结和 PPT 共用配图。
- `05_中间素材/`：章节文本、清洗结果和导出预览。

## 工程位置

- `00_项目说明/生成脚本/`：Python、Node 和 PowerShell 脚本入口。
- `package.json` 与 `package-lock.json`：Node 依赖说明。

## 推荐阅读顺序

1. 先看 `00_项目说明/README.md`。
2. 再看 `01_阅读报告/` 和 `02_关键知识总结/`。
3. 需要授课或演示时进入 `03_PPT/`。
